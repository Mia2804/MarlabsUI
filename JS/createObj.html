There are four methods to create an object:
1. Object Literals:An object literal, also called an object initializer, is a comma-separated set of paired names and values.
2. New operator or constructor: If you call a function using a new operator, the function acts as a constructor and returns an object.
3. Object.create methos: You can also create new objects using the Object.create() method, which allows you to specify the prototype object and the properties.
